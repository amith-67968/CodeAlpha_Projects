# CodeAlpha Python Programming Tasks

This repository contains my completed Python Programming Internship tasks for CodeAlpha.

## Tasks Completed

### 1. Hangman Game
A text-based word guessing game where the player tries to guess a hidden word one letter at a time.

#### Features
- Random word selection
- Limited incorrect attempts
- Displays guessed letters
- Console-based gameplay

#### Concepts Used
- Random Module
- Loops
- Conditional Statements
- Strings
- Lists

---

### 2. Basic Chatbot
A simple rule-based chatbot that responds to predefined user inputs.

#### Features
- Greeting responses
- Common question handling
- Exit command support
- Interactive conversation

#### Concepts Used
- Functions
- Dictionaries
- Loops
- Conditional Statements
- User Input Handling

---

### 3. Stock Portfolio Tracker
A Python application that calculates the total value of stock investments based on user input.

#### Features
- Stock price tracking
- Quantity input
- Portfolio value calculation
- Save portfolio summary to TXT file

#### Concepts Used
- Dictionaries
- File Handling
- Loops
- Arithmetic Operations
- User Input Processing

---
